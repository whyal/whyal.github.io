# Yong Lun | Full-Stack Software Engineer

A personal website. Built with **Next.js 16.2.1** and **TypeScript**.

## Deployment

This site is statically exported and hosted via **GitHub Pages**.

## Development

`   npm install  npm run dev   `

## Build & Export

`   npm run build   `

The static files are generated in the /out directory, which is automatically deployed via GitHub Actions.
